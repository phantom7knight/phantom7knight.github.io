# phantom7knight.github.io

Personal portfolio of Rohit Tolety — game / graphics / tools programmer.

Live at **https://phantom7knight.github.io**

Plain HTML/CSS/JS, no build step. Served by GitHub Pages from the `main` branch.

## Editing

- Pages: `*.html` at the root, project pages under `projects/`
- Theme: `css/theme.css` (all colors are CSS variables at the top)
- Homepage typing animation: `js/terminal.js` + the JSON block in `index.html`
- To update the resume, replace `assets/files/resume.pdf`

## Local preview

```sh
python -m http.server 8000
# open http://localhost:8000
```
