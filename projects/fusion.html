Fusion — rendering engine

C++D3D12Vulkan NVRHIHLSLImGui CMakeactive development

Fusion is my current main project: a rendering engine written in modern C++ on top of NVRHI, NVIDIA's rendering hardware interface, which lets the same code target both Direct3D 12 and Vulkan. The goal is a playground where I can implement rendering techniques properly — not just get them on screen once, but build them into a reusable engine.

The engine is organized as a core library plus a set of applications, each one a stepping stone: Init (windowing, first triangle, model loading), Deferred (full deferred-shading pipeline), MaterialsPlayground (interactive material editing) and MeshShaders.

features

  • Deferred shading with a full G-buffer — depth, diffuse, normals, specular and transmission targets.
  • Deferred lighting resolved in a compute shader rather than on the graphics pipeline.
  • PBR materials with transmission support (glass-like dragons below).
  • ImGui-based material editor for live-tweaking material parameters on loaded models.
  • Config-driven shader compilation pipeline (HLSL).
  • In progress: image-based lighting (IBL) — BRDF LUT, irradiance and prefiltered specular cubemaps.
