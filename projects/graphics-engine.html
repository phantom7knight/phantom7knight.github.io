Graphics Engine (OpenGL)

OpenGLC++Compute shaders Solo project

A solo forward-rendering engine built with OpenGL and C++ — the project where I first went deep on physically based lighting and GPU compute.

features

  • Forward renderer with a microfacet BRDF lighting model.
  • Bloom post-processing with Gaussian blur.
  • GPU particle system driven by compute shaders — 2 million+ particles on screen.
